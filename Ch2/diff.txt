* 比對檔案
** git diff
* git tool: kdiff3 (for windows)
** git config --global diff.tool kdiff3
** git config --global difftool.kdiff3.cmd \ "'C:/xxxxx/kdiff3.exe' \"\$LOCAL\" \"\$REMOTE\""
** git config --global difftool.prompt false
* git tool: diffmerge
** http://twobitlabs.com/2011/08/install-diffmerge-git-mac-os-x/

git diff 不同指令在 實體 <-> 索引 <--> Repo 的關係
* git diff filename
* git difftool filename
=> 實體 <-> 索引 <--> Repo

git diff 識別碼1 識別碼2 filename
=> Repo1 <--> Repo2

git diff --no-index filename1 filename2
=> 實體檔案1 <-> 十
